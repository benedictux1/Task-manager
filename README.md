# Task Manager

A three-view task and project management web application with Apple-inspired minimalist design.

## Features

- **Project View**: Manage projects with rich-text notes and associated tasks
- **Task View**: Unified view of all tasks with filtering and sorting
- **Settings**: Customize task types and statuses with color coding
- Rich text editing for project notes
- Collapsible sidebar
- Status-colored task checkboxes

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Running Locally

Start the development server:
```bash
npm run dev
```

The app will open automatically at `http://localhost:3000`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
